Cyborg Institute Issue Tracking

In most cases, Cyborg Institute projects use the Bugs Everywhere issue tracker, which stores issues in git repository with the source for each project. Using these issues, we generate the following issue displays linked below.

To submit an issue on a Cyborg Institute project, you can either install Bugs Everywhere clone the repository, file a new issue, and submit a pull request, or you can send an email as described on the contribution page for the project linked below. Instructions for pull requests are also below.
